The Intel Core 2 Duo, introduced in 2006, was a groundbreaking 64-bit dual-core processor family that significantly boosted performance over previous Pentium models using the efficient Core microarchitecture.

These processors, often utilizing Socket 775, are largely obsolete for modern gaming but remain suitable for basic retro computing, lightweight tasks, and older operating systems like Windows XP.
